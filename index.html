GU concentration test
Out of 100 two-digit numbers, identify as quickly as possible the 25 numbers whose first digit is even and whose second digit is odd

Further information on the GU test

Objective:
A version of the GU is presented that can be used by interested scientists for research purposes.

GU - Testing

The test person must complete 2 rounds and receives feedback on their results at the end. The test results can be copied and pasted into a spreadsheet program.

GU with timer - Testing

The test person must complete 2 rounds and receives feedback on their results at the end. The test results can be copied and pasted into a spreadsheet program.
The test contains a running timer that puts the participants under stress.

UG - Testing

Parallel version to the GU. Here, the test person must click on all numbers whose first digit is odd and whose second digit is even. Procedure analogous to GU testing. No empirical study has been conducted on this version to date.

UG with timer - Testing

Parallel version to the GU. Here, the test person must click on all numbers whose first digit is odd and whose second digit is even. Procedure analogous to GU testing. No empirical study has been conducted on this version to date.
The test contains a running timer that puts the participants under stress.

GG-UU testing
Another variant as a possible parallel test. Here the test person must click on numbers that contain either only even or only odd digits. There is no empirical study on this version to date.

Sending tests

After sending the tests, you can edit them on your own computer without a network connection. It is then also possible to change the number of test runs in order to adapt the test time to the desired or tolerable reliability. For example, to reduce the number of test runs to 1, simply change the line maxtrial=2 to maxtrial=1 in the JavaScript of the gu.html or ug.html files.

If you have some knowledge of JavaScript and php, you can use the files to create an online survey relatively easily. The programs also serve as possible templates for constructing several other test variants, e.g. relation.

Note: there is already a GU-Test on the Internet that requires three test runs and provides the student with a very detailed test interpretation. The version presented here is a slightly improved version in terms of the test instructions, which does not provide a test interpretation but does report all results.
